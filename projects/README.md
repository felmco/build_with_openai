### [1. Auto Researcher Agent](./01-auto-researcher/README.md)
**Level**: Intermediate
**Tech**: `o1-mini` (Reasoning), Agents SDK, Function Calling
**Goal**: Build an autonomous agent that can research a broad topic, break it down into sub-questions, and compile a comprehensive summary report.
**What you'll learn**:
- How to implement agentic reasoning loops.
- Managing long-running tasks.
- Structuring complex outputs.

### [2. Vision Sales Assistant](./02-vision-assistant/README.md)
**Level**: Beginner-Intermediate
**Tech**: `gpt-4o` (Vision), Structured Outputs
**Goal**: Create a tool that analyzes product photos and automatically generates SEO-friendly descriptions, social media captions, and pricing estimates.
**What you'll learn**:
- processing images with GPT-4o.
- Extracting structured data from visual inputs.
- Generating multi-channel content.

### [3. Voice Ops Commander](./03-voice-commander/README.md)
**Level**: Advanced
**Tech**: Whisper (ASR), TTS-1 (Audio), Realtime API concepts
**Goal**: Build a voice-controlled interface for your computer that can listen to commands, execute system scripts, and speak back the results.
**What you'll learn**:
- Handling audio input/output.
- Building voice-to-action pipelines.
- Creating a natural conversational interface.
